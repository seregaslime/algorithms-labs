## Лабораторная 1 — Поиск в массиве

Реализованы 4 алгоритма поиска с замером количества сравнений и времени выполнения:

- **Better_Linear_Search** — линейный поиск с двумя счётчиками (сравнение элемента с ключом, индекса с длиной массива)
- **Sentinel_Linear_Search** — линейный поиск с барьером (sentinel) — без проверки границ в цикле
- **Ordered_Array_Search** — поиск в упорядоченном массиве с ранним выходом
- **Binary_Search** — классический бинарный поиск

**Сборка:**
```bash
g++ -std=c++11 -O2 lab1/alg.cpp -o lab1/alg && ./lab1/alg
```

## Лабораторная 2 — Сортировка массива

Реализованы 2 алгоритма сортировки с замером сравнений, обменов и времени. Проверка на трёх типах последовательностей: случайная, возрастающая, убывающая.

- **Selection_Sort** — сортировка выбором (поиск минимума и обмен)
- **Quick_Sort** — быстрая сортировка (схема Lomuto, опорный — последний элемент)

**Сборка:**
```bash
g++ -std=c++11 -O2 lab2/lab2.cpp -o lab2/lab2 && ./lab2/lab2
```

## Лабораторная 3 — Линейные связные списки

Односвязный список целых чисел с интерактивным меню. Список хранит только уникальные значения — при обнаружении дубликата при создании список уничтожается, при добавлении дубликат отклоняется.

**Операции:**
- Создание списка (ввод до 0, элементы добавляются в начало)
- Вывод всех элементов
- Удаление списка
- Поиск максимального элемента и его индекса
- Добавление элемента в конец
- Удаление элемента по индексу

**Сборка:**
```bash
g++ -std=c++11 -O2 lab3/lab3.cpp -o lab3/lab3 && ./lab3/lab3
```

## Лабораторная 4 — Бинарные деревья поиска

Два этапа: демонстрация операций BST и анализ высоты деревьев (обычное BST vs рандомизированное BST).

### Stage 1 — Базовые операции
- Вставка, поиск, удаление (3 случая: лист, один потомок, два потомка)
- Вывод дерева, подсчёт высоты и количества узлов
- Inorder-обход (отсортированная последовательность)
- **Вариант 8:** подсчёт нечётных узлов, имеющих два поддерева

### Stage 2 — Сравнение высот
- Для случайной и отсортированной последовательностей (25–300 ключей)
- Построение обычного BST и рандомизированного BST
- Сравнение высот в таблице

**Сборка:**
```bash
g++ -std=c++11 -O2 lab4/main.cpp lab4/tree.cpp -o lab4/main && ./lab4/main
```
